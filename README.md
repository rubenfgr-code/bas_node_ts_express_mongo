# Root project with NodeJS, TypeScript, Express and MongoDB

## Start project

```sh
git clone git@github.com:rubenfgr-doc/bas_node_ts_express_mongo.git
npm install
```

## Scripts

```sh
npm run test
npm run start
npm run start-ts
npm run start-nodemon
```
